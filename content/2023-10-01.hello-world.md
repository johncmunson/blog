# Paragraphs & Emphasis

Multiple paragraphs keep thoughts organized. You can _italicize_, **bold**, combine **_both_**, or deploy ~~strikethrough~~ for edits. Inline code like `const hello = "world";` draws attention to syntax, while automatic links such as https://nextjs.org appear without extra fuss.

### Lists You Can Nest

- Bullet lists explain things succinctly.
  - Sub-bullets let you dive deeper.
  - Mix and match text, `inline code`, and **emphasis**.
- Task lists double as simple checklists:
  - [x] Write the Hello World post
  - [ ] Ship it to production

1. Ordered lists keep steps clear.
2. They can include paragraphs, links, and even multi-line content.
3. Combine them with sublists for detailed procedures.

## Quick Reference

| Feature    | Example                         | Notes                                |
| ---------- | ------------------------------- | ------------------------------------ |
| Headings   | `#`, `##`, `###`                | Use to structure long posts.         |
| Emphasis   | `_italic_`, `**bold**`          | Combine for stylistic flair.         |
| Links      | `[Next.js](https://nextjs.org)` | Autolinks also supported.            |
| Images     | `![alt](path)`                  | See the kitty-cat and doggo above.   |
| Code       | `` `inline` `` / fenced blocks  | Triple backticks for multi-line.     |
| Task Lists | `- [x] done`                    | Render as checkboxes.                |
| Tables     | Pipes + dashes                  | Align columns with colons if needed. |
